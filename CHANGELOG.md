# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2024-10-09

### Added
- Initial release of TypeScript Value Duplicates Finder
- Advanced scanning for TypeScript interfaces, types, and enums
- Support for complex nested types and generic types
- Configurable scanning paths and exclusion patterns
- Multiple output formats (console, JSON, both)
- Hebrew and English bilingual interface
- Smart type normalization for accurate duplicate detection
- Severity scoring for prioritizing fixes
- Automatic base name suggestions for refactoring
- CLI interface with help and configuration options
- Comprehensive test suite
- Example files for testing and demonstration
- Universal project support - works with any TypeScript project
- Smart filtering to ignore test files and declaration files
